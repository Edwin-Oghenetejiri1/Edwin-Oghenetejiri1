### 🚀 Featured Project

#### 🕹️ RetroGame Shop — End-to-End DevOps on AWS EKS
> Microservices e-commerce platform built with 5 languages, deployed on AWS EKS using GitOps principles

- ✅ **6 Microservices** — Node.js, Go, Python, Java, C#
- ✅ **Infrastructure as Code** — Terraform provisions VPC, EKS, IAM
- ✅ **CI/CD Pipelines** — GitHub Actions with Trivy security scanning
- ✅ **GitOps** — ArgoCD syncs manifests repo to EKS automatically
- ✅ **Observability** — Prometheus + Grafana monitoring
- ✅ **Auto Scaling** — Karpenter for dynamic node provisioning
